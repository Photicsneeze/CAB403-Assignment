# CAB403 - Assignment

Open terminal and navigate to home directory of project (where makefile is located).

To compile, run command:
	make

To run the server, execute this command in terminal:
	./bin/server port_number

Where port_number is the port on which the server is being hosted (default is 12345).

To connect a client,execute this command in terminal:
	./bin/client localhost port_number
	
Where port_number is the same port used in the server command.
